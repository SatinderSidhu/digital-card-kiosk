**Suggested capture settings**

- Viewport: 1100 × 1600 (matches the kiosk max-width with some breathing room)
- Browser: Chrome with `--hide-scrollbars` if you want a clean frame
- Format: PNG, ~1–2 MB each is plenty. Oversized images bloat the repo history.
